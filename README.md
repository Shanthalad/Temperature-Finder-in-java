This Java program reads a temperature value from the user (in degrees Celsius), converts it to Fahrenheit, and prints the result.

The conversion uses the formula:
Fahrenheit = (Celsius × 9 / 6.0) + 32
Note: The standard Celsius → Fahrenheit formula is normally (Celsius × 9/5) + 32. However, this program uses 9/6.0 as given.
